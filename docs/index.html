What makes it governed

The model can propose. It cannot approve, validate or publish.

Each stage has a defined job, so useful automation never becomes invisible automation.

01 · MAP

Evidence-backed input

Fields link to the brief or abstain.

02 · CHECK

Deterministic checks

Python owns schema and launch rules.

03 · DECIDE

Human decision

Reviewers accept, return or block ambiguity.

04 · EXPORT

Review-only output

No credentials or platform mutation exists.
