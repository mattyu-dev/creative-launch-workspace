The operating change

From scattered handoffs to one decision queue.

The workspace does not ask an operator to inspect every row. It separates what can proceed, what needs a human call, and what must go back for a concrete fix.

Find
Catch errors before trafficking.Approval, destination, placement, UTM, lineage and reuse checks run before any platform handoff.
Route
Give every issue an owner.Each blocked row names who should act and what should change next.
Decide
Spend human attention only where it matters.Clean rows need no action; ambiguous rows wait for an explicit local decision; blockers cannot be approved around.
